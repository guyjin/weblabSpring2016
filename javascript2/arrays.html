Arrays

array.toString()

For Array objects, the toString method joins the array and returns one string containing each array element separated by commas. For example, the following code creates an array and uses toString to convert the array to a string. 

            
                var fruit = ['apples','oranges','bananas'];
                var myfruit = fruit.toString();

array.valueOf()

The valueOf() method returns the array. 

            
                var fruit = ['apples','oranges'];
                fruit.valueOf();

array.slice()

The slice() method returns a shallow copy of a portion of an array into a new array object. 

            
                var fruit = ['apples','oranges','bananas'];
                fruit.slice(0,1)

array.concat()

The concat() method returns a new array comprised of the array on which it is called joined with the array(s) and/or value(s) provided as arguments. 

            
                var fruit = ['apples','oranges','bananas'];
                var veggies = ['peas','carrots','onions'];
                var healthy = fruit.concat(veggies);
                console.log(healthy);

array.sort()

The sort() method sorts the elements of an array in place and returns the array. The sort is not necessarily stable. The default sort order is according to string Unicode code points. 


            var fruit = ['cherries', 'apples', 'bananas'];
            fruit.sort(); // ['apples', 'bananas', 'cherries']

array.reverse()

Reverses the order of the elements of an array in place â€” the first becomes the last, and the last becomes the first. 


            var fruit = ['apples','oranges','bananas'];
            fruit.reverse()

array.push()

Add to the end of an Array 


            var fruit = ['apples','oranges','bananas'];
            fruit.push('grapes');
            console.log(fruit);

array.length

The length property represents an unsigned, 32-bit integer that is always numerically greater than the highest index in the array. 


            var fruit = ['apples','oranges','bananas'];
            fruit.lenth;

array.shift()

The shift() method removes the first element from an array and returns that element. This method changes the length of the array. 


            var fruit = ['apples','oranges','bananas'];
            var newFruit = fruit.shift();
            console.log(newFruit);

array.unshift()

The unshift() method adds one or more elements to the beginning of an array and returns the new length of the array. 


            var fruit = ['apples','oranges','bananas'];
            fruit.unshift('grapes');
            console.log(fruit);

array.push()

The push() method adds one or more elements to the end of an array and returns the new length of the array. 


            var fruit = ['apples','oranges','bananas'];
            fruit.push('grapes');
            console.log(fruit);

array.pop()

The pop() method removes the last element from an array and returns that element. 


            var fruit = ['apples','oranges','bananas'];
            fruit.pop();  // bananas

array.splice()

The splice() method changes the content of an array by removing existing elements and/or adding new elements. 


            var fruit = ['apples','oranges','bananas'];
            var oldFruit = fruit.splice(1,1,'grapes')
            console.log(fruit, oldFruit)

array.join()

The join() method joins all elements of an array into a string. 


            var fruit = ['apples','oranges','bananas'];
            var myFruit1 = fruit.join();
            console.log(myFruit1);
            var myFruit2 = fruit.join(' + ');
            console.log(myFruit2);

array.map()

The map() method creates a new array with the results of calling a provided function on every element in this array. 


            var fruit = ['apples','oranges','bananas'];
            var fruitLengths = fruit.map(function(obj){
                var l = obj.length;
                return l;
            });
            console.log(fruitLengths)

array.reduce()

The reduce() method applies a function against an accumulator and each value of the array (from left-to-right) to reduce it to a single value. 


            var fruit = ['apples','oranges','bananas'];
            var fruitLengths = fruit.reduce(function(total,num) {
                return total + num;
            });
            console.log(fruitLengths); //applesorangesbananas

array.filter()

The filter() method creates a new array with all elements that pass the test implemented by the provided function. 


            function isBigEnough(value) {
              return value.length >= 7;
            }
            var filtered = ['apples','oranges','bananas'].filter(isBigEnough);
            console.log(filtered); //["oranges", "bananas"]
